# Политика конфиденциальности приложения "ИМПУЛЬС"
**Последнее обновление: 14 февраля 2026 г.**

Разработчик (далее "Мы", "Разработчик") уважает конфиденциальность пользователей приложения "ИМПУЛЬС" (далее "Приложение").

## 1. Какие данные мы собираем

Приложение не требует регистрации аккаунта на серверах Разработчика. Мы обрабатываем следующие типы данных:

1.1. **Технические данные:**
*   Информация об устройстве (модель, версия Android).
*   Логи сбоев (через системные отчеты Android/RuStore) для исправления ошибок.

1.2. **Данные о покупках:**
*   Для предоставления доступа к Premium-подписке мы используем RuStore SDK (VK). Ваша платежная информация обрабатывается исключительно на стороне RuStore, мы получаем только подтверждение статуса подписки (активна/неактивна).

1.3. **Данные камеры:**
*   Приложение использует камеру устройства для распознавания поз тела (технология Google ML Kit Pose Detection).
*   Изображения обрабатываются **исключительно локально** на устройстве в реальном времени и **НЕ сохраняются и НЕ передаются** на серверы.
*   Камера используется только во время выполнения физических упражнений и только с явного разрешения пользователя.

1.4. **Статистика использования приложений:**
*   Приложение запрашивает доступ к статистике использования (UsageStats) для определения, какое приложение открыто на переднем плане.
*   Эти данные обрабатываются **исключительно локально** и используются только для функции контроля экранного времени.
*   Мы НЕ собираем и НЕ передаем историю использования приложений.

1.5. **Локальные данные:**
*   Настройки приложения, статистика тренировок, список заблокированных приложений и баланс экранного времени хранятся исключительно в памяти вашего устройства.

## 2. Как мы используем данные

Мы используем данные исключительно для:
*   Обеспечения работоспособности функций приложения (контроль экранного времени, физические и когнитивные упражнения).
*   Проверки наличия лицензии (Premium-доступа).
*   Улучшения стабильности приложения (анализ сбоев).

## 3. Передача данных третьим лицам

Мы не продаем и не передаем ваши личные данные третьим лицам.

В приложении используются сторонние сервисы:
*   **RuStore SDK:** для обработки встроенных покупок и проверки обновлений. Политика конфиденциальности VK (RuStore): https://rustore.ru/legal/privacy
*   **Google ML Kit:** для локального распознавания поз тела. Данные НЕ покидают устройство. Условия: https://developers.google.com/ml-kit/terms

## 4. Удаление данных

Вы можете в любой момент удалить все локальные данные, воспользовавшись функцией "Сбросить данные" в настройках приложения или просто удалив Приложение с устройства.

## 5. Соответствие законодательству

Обработка данных осуществляется в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».
